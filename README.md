## Projeto

O Formulário de Matrícula Escolar é uma página desenvolvida com foco em praticar estruturação com HTML e estilização com CSS, organizando informações em uma interface clara e bem distribuída. O projeto simula um formulário de matrícula para escola infantil, reunindo campos de cadastro, seleção de opções e validação visual da interface.

## Tecnologias utilizadas

- HTML
- CSS
- Git
- GitHub
- Figma

## Funcionalidades

- Formulário de matrícula com campos de informações da criança e do responsável
- Organização visual dos campos em seções para facilitar o preenchimento
- Área de upload de arquivo para certidão de nascimento
- Opções de seleção para turno de estudo e atividade esportiva
- Estrutura visual com destaque para mensagens de validação
- Layout organizado em blocos de conteúdo

## Aprendizados

Neste projeto, pratiquei:

- estruturação de formulários com HTML
- estilização e organização visual com CSS
- composição de layout com múltiplas seções
- organização de conteúdo para melhorar a experiência de preenchimento
- criação de interface com foco em clareza visual
- uso de Git e GitHub para versionamento

## Créditos

Projeto desenvolvido com base no conteúdo do FullStack, da Rocketseat, com personalizações aplicadas durante a prática.
